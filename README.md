# AI4DirtyWork

Create new skills, improve existing skills, and measure skill performance. Use when users want to create a skill from scratch, update or optimize an existing skill, run evals to test a skill, or benchmark skill performance with variance analysis.

# Skills
- [skill-creator](./skills/skill-creator): Create, evaluate, and iterate on Claude Code skills
- [one-page-think](./skills/one-page-think): Toyota One-Page minimal thinking method

# Try in Claude Code

## Install via Plugin Marketplace
```
/plugin marketplace add ZhengSJCode/AI4DirtyWork
```

Then:
1. Select `Browse and install plugins`
2. Select `AI4DirtyWork`
3. Select the skill you want
4. Select `Install now`

## Direct Install
```
/plugin install skill-for-dirtywork@AI4DirtyWork
```

After installing, mention the skill in your prompt. For example: "Use the one-page-think skill to analyze this problem."
